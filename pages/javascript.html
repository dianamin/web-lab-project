Framework-uri și librării de Javascript

AngularJS

Ce știe să facă?

Framework-ul AngularJS te ajută să extinzi vocabularul HTML-ului pentru aplicația ta. Se bazează pe „controllere” - cu AngularJS, descrii comportamentul elementelor DOM pe care le utilizezi într-un mod elegant în Javascript.

AngularJS se autointitulează „Superheroic JavaScript MVW Framework”. Mai multe detalii aici.

jQuery

Ce știe să facă?

jQuery este o librărie foarte utilă pentru:

  • selectarea elementelor HTML dintr-un document
  • includerea unor noi elemente HTML dinamic în pagină
  • gestionarea evenimentelor
  • animații
  • AJAX

Mai multe detalii aici.

Polymer

Ce știe să facă?

Polymer este o librărie cu ajutorul căreia îți poți crea propriile elemente HTML. Poți crea un șablon HTML stilizat cu CSS și îi poți descrie comportamentul utilizând metode și proprietăți în Javascript.

Are deja o serie de elemente create și care folosesc Material Design.

Mai multe detalii aici.

React

Ce știe să facă?

Un termen important pentru a înțelege cum merge librăria React este paradigma de programare MVC - Model View Controller. Angular se apropie de paradigma MVC, iar React este doar V din MVC. Practic, este foarte util pentru afișarea datelor.
