Framework-uri și librării de CSS

Bootstrap

Ce știe să facă?

Bootstrap este probabil cel mai popular framework de HTML, CSS și Javascript. Cu Bootstrap Web Development-ul devine mult mai rapid. Este foarte ușor de folosit și este potrivit pentru ecrane de orice formă și de orice dimensiuni.

Pe partea de CSS, Bootstrap stilizează elementele HTML fundamentale implementând clase CSS extensibile și introduce un sistem de tip Grid.

Ce e Material Design?
