Ce înseamnă Frontend?

Un site web este, în general, format din două părți:

Frontend - Când discutăm despre Frontend, ne gândim la HTML, la CSS și la Javascript, la partea site-ului cu care interacționează utilizatorul și care afișează conținutul.
Backend - Câteva cuvinte în strânsă legătură cu termenul de Backend sunt PHP, Ruby, Python, server, bază de date. Rolul principal al Backend-ului este cel de management de conținut.

Framework? Library?

Framework
  • Tradus din engleză: cadru, schelet, ramă, arhitectură, construcție.
  • Termenul nu este numai pentru Web Development.
  • Se referă la cod ce furnizează un set de funcționalități generice care te ajută să scrii o aplicație fără a reinventa roata.
  • Fixează modul în care codul trebuie să fie structurat.
Library
  • La fel ca la Framework, termenul nu este numai pentru Web Development.
  • La fel ca la Framework, se referă la cod ce furnizează un set de funcționalități generice care te ajută să scrii o aplicație fără a reinventa roata.
  • Spre deosebire de Framework, nu fixează modul în care codul trebuie să fie structurat.
